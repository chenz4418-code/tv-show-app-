## 5. 故障排除

如果海报或头像显示失败，可能有以下原因：

1. **文件名不正确**：请确保文件名与上表完全一致（区分大小写）
2. **文件格式不支持**：请使用 JPG、JPEG 或 PNG 格式
3. **文件路径错误**：请确保图片放置在正确的目录中（海报放在 `posters`，头像放在 `avatars`）
4. **文件损坏**：请检查图片文件是否完整可打开

如果仍然无法解决问题，应用程序会自动显示默认的 SVG 海报或 SVG 头像作为备用。

## 6. 自定义更多资源

### 自定义海报
如果您想为其他剧集添加本地海报，可以在 `app.py` 文件中修改数据库部分：

```python
"剧集名称": {
    "poster": get_local_poster("posters/文件名.jpg"),
    # 其他信息
}
```

### 自定义头像
如果您想为其他角色添加本地头像，可以在 `app.py` 文件中修改数据库部分的 nodes 列表：

```python
"nodes": [
    ("角色名称", get_local_avatar("avatars/文件名.jpg")),
    # 其他角色
],
```
